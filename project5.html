Personal holiday project developed in Python that consists of employee registration and payment generation through a layout developed in Tkinter. The program features data display in Treeviews, backend queries using SQLite3 database, and PDF printing with ReportLab. All frames were designed manually, and the program works locally.

What I learned or improved:
1 - OOP in Python.
2 - Screen design in Tkinter.
3 - Form creation.
4 - Exported PDF Files

Development of a PHP/JavaScript web app for a IT store using MySQL database. Features include CRUD operations for clients, technicians, materials, catalog, services, and production. The program also features a Dashboard that reacts to status changes, administrator permissions, serial number generator, and hour calculation. Design was stylized using CSS, aiming for optimized access to functionalities.

What I learned or improved:
1 - Building a database from scratch based on an Entity-Relationship diagram.
2 - Defining all FK relationships optimizing the use of SQL queries.
3 - Web Design with CSS, responsive divs, and respect for UX.
4 - Integrating JavaScript with PHP (It's truly a perfect synergy).
5 - Using GET and POST methods to move data across pages.
6 - Working with databases in PHP backend.
7 - CRUD methods implementation
8 - OOP PHP
9 - Mysql Server

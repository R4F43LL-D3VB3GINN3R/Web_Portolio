Personal project developed in ABAP that involves the registration and generation of payments for employees using a layout created in Module Dynpro.
The project was implemented in SAP GUI and structured with OOP concepts, with methods applied in the PAI (Process Interface Program).

What I learned or improved:
1 - Creation, definition, and implementation of methods in ABAP.
2 - Development and implementation of RFC (Remote Function Calls) Functions.
3 - Display of ALV (ABAP List Viewer) reports.
4 - Functions to export ALV data to CSV.
5 - Execution of SQL queries with ABAP syntax.
6 - Screen design using the Screen Painter.
7 - Code modularization using Includes.
